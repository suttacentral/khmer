ឧទ្ទាន គឺបញ្ជីរឿង នៃមហាយមកវគ្គ ទី ៤ នេះ គឺនិយាយពីព្រះមានព្រះភាគ គង់ក្នុងរោងដែលកដោយឥដ្ឋ ក្នុងព្រៃគោសិង្គសាលវ័ន ( មានក្នុងចូឡគោសិង្គសាល សូត្រ និងមហាគោសិង្គសាលសូត្រ ) ១ និយាយពីធម៌ប្រៀបដោយគោបាល គួររក្សា គោបាននិងមិនបាន ( មានក្នុងមហាគោបាលសូត្រ ) ១ និយាយអំពីគោបាលមានប្រាជ្ញា ម្តងទៀត ( មានក្នុងចូឡគោបាលសូត្រ ) ១ និយាយពីព្រះមានព្រះភាគ ទ្រង់គង់នៅក្នុង ក្រុងវេសាលី ( មានក្នុងចូឡសច្ចកសូត្រ ) ១ និយាយពីព្រះមានព្រះភាគមានព្រះភក្ត្រ ជ្រះថ្លាជាដើម ( មានក្នុងមហាសច្ចកសូត្រ ) ១ ។ និយាយពីព្រះឥន្រ្ទាធិរាជ ប្រកបដោយ តណ្ហា ( មានក្នុងចូឡតណ្ហាសង្ខយសូត្រ ) ១ និយាយពីសាតិភិក្ខុកេវដ្តបុត្ត ( មានក្នុងមហាតណ្ហាសង្ខយសូត្រ ) ១ និយាយពីពាក្យថាសមណៈ មានក្នុង មហាអស្ស-បុរសូត្រ ១ និយាយពីសមណៈទុកផ្នួងសក់ មានក្នុងចូឡអស្សរសូត្រ ១ ។
