បិយជាតិកសូត្រ ទី ៧

២៩៦ខ្ញុំបានស្ដាប់មកយ៉ាងនេះ ។ សម័យមួយ ព្រះមានព្រះភាគ ទ្រង់សម្រេចព្រះឥរិយាបថ ក្នុងជេតវនារាម របស់អនាថបណ្ឌិតសេដ្ឋី ជិតក្រុងសាវត្ថី ។ សម័យនោះឯង គហបតីម្នាក់មានកូនប្រុសតូចតែមួយ ជាទីស្រឡាញ់ពេញចិត្ត បានធ្វើមរណកាលទៅ ។ ព្រោះកូននោះធ្វើកាលកិរិយាទៅ ការងារទាំងឡាយក៏មិនប្រាកដ ភត្តាហារក៏មិនប្រាកដ ( ដល់គាត់ ) ឡើយ ។ គាត់តែងតែទៅកាន់ឈាបនដ្ឋានយំរកថា ឪកូនតូចថ្លើមថ្លៃ អ្នកនៅក្នុងទីណា ឪកូនតូចថ្លើមថ្លៃ អ្នកនៅក្នុងទីណា ។ គ្រាក្រោយមក គហបតីនោះ ចូលទៅគាល់ព្រះមានព្រះភាគ លុះចូលទៅដល់ហើយ ក៏ថ្វាយបង្គំព្រះមានព្រះភាគ រួចអង្គុយក្នុងទីដ៏សមគួរ ។

២៩៧លុះគហបតី អង្គុយក្នុងទីដ៏សមគួរហើយ ព្រះមានបុណ្យទ្រង់ត្រាស់ដូច្នេះថា ម្នាលគហបតី អ្នកតាំងនៅក្នុងចិត្តរបស់ខ្លួនគឺនៅរស់ទេ តែហាក់ដូចជាមិនមានឥន្រ្ទិយគឺមនោវិញ្ញាណ ( ព្រោះ ) ឥន្រ្ទិយរបស់អ្នកប្រែសម្បុរប្លែក ។ គហបតី ទូលថា ឪព្រះអង្គដ៏ចំរើនឥន្រ្ទិយរបស់ខ្ញុំព្រះអង្គ នឹងមិនប្រែពណ៌សម្បុរប្លែកណាក៏បាន បពិត្រព្រះអង្គដ៏ចំរើន ព្រោះខ្ញុំព្រះអង្គមានកូនប្រុសតូចតែមួយ ជាទីស្រឡាញ់ពេញចិត្ត បានធ្វើមរណកាលទៅ ព្រោះវាធ្វើកាលកិរិយាទៅ ការងារក៏មិនប្រាកដ ភត្តាហារក៏មិនប្រាកដ ( ដល់ខ្ញុំព្រះអង្គ ) សោះ ខ្ញុំព្រះអង្គនោះ តែងទៅកាន់ ឈាបនដ្ឋាន ហើយយំរកថា ឪកូនតូចមាសថ្លៃអ្នកនៅក្នុងទីណា ឪកូនតូចមាសថ្លៃ អ្នកនៅក្នុងទីណា ។ ព្រះមានបុណ្យត្រាស់ថា ដំណើរនុ៎ះយ៉ាងនោះមែន គហបតី ដំណើរនុ៎ះ យ៉ាងនោះមែន គហបតី ពិតមែន សេចកី្ដសោកស្ដាយ ខ្សឹកខ្សួលសេចកី្ដទុក្ខ តូចចិត្ត ការចង្អៀល់ចិត្ត តែងកើតមកអំពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើត ។ គហបតីក្រាបបង្គំទូលថាបពិត្រព្រះអង្គដ៏ចំរើន ធម្មតាហេតុនៃសេចកី្ដស្រឡាញ់នោះឯង នឹងមានដូច្នោះមែន សេចកី្ដសោកស្ដាយ ខ្សឹកខ្សួល សេចកី្ដទុក្ខ តូចចិត្ត ការចង្អៀតចង្អល់ចិត្ត កើតមកអំពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើតដែរ បពិត្រព្រះអង្គដ៏ចំរើន តែបើតាមដែលពិតនោះ សេចកី្ដត្រេកអរនឹងសេចកី្ដរីករាយ តែងកើតមក អំពីសេចក្ដីស្រឡាញ់ មានសេចក្ដីស្រឡាញ់ជាដែនកើត១ ។ ខណៈនោះ គហបតីនោះ មិនត្រេកអរ មិនឃាត់ហាមនូវភាសិត របស់ព្រះមានបុណ្យទេ ក៏ក្រោកចាកអាសនៈចេញទៅ ។

២៩៨សម័យនោះ មានពួកអ្នកលែងល្បែងភា្នល់ច្រើននាក់កំពុងលេងដោយល្បែងភ្នាល់ទាំងឡាយ ក្នុងទីមិនឆ្ងាយប៉ុន្មានអំពីព្រះមានបុណ្យ ។ ចំណែកគហបតីនោះ ក៏ចូលទៅរកពួកអ្នកលេងល្បែងភ្នាល់នោះ លុះចូលទៅដល់ហើយ ក៏និយាយទៅនឹងអ្នកលេងល្បែងភ្នាល់នោះដូច្នេះថា នែអ្នកដ៏ចំរើនទាំងឡាយ អម្បាញ់មិញនេះ ខ្ញុំបានចូលទៅគាល់ព្រះសមណគោតម លុះចូលទៅដល់ហើយ ក៏ថ្វាយបង្គំព្រះសមណគោតម រួចអង្គុយក្នុងទីដ៏សមគួរ នែអ្នកដ៏ចំរើនទាំងឡាយលុះខ្ញុំអង្គុយក្នុងទីដ៏សមគួរហើយ ព្រះសមណគោតមទ្រង់ត្រាស់ដូច្នេះថាម្នាលគហបតី អ្នកឋិតនៅក្នុងចិត្តរបស់ខ្លួនគឺរស់នៅទេ តែហាក់ដូចជាមិនមានឥន្រ្ទិយ ( ព្រោះ ) ឥន្រ្ទយរបស់អ្នកប្រែសម្បុរប្លែកដូច្នេះ នែអ្នកដ៏ចំរើនទាំងឡាយ កាលព្រះសមណគោតមត្រាស់យ៉ាងនេះ ខ្ញុំក៏បានតបទៅនឹងព្រះសមណគោតមវិញថា ឪព្រះអង្គដ៏ចំរើន ឥន្រ្ទយរបស់ខ្ញុំព្រះអង្គនឹងមិនប្រែពណ៌សម្បុរប្លែកណាក៏បាន បពិត្រព្រះអង្គដ៏ចំរើនព្រោះខ្ញុំព្រះអង្គមានកូនប្រុសតូចតែមួយ ជាទីស្រឡាញ់ពេញចិត្តបានធ្វើមរណកាលទៅ ព្រោះតែវាធ្វើកាលកិរិយាទៅ ការងារក៏មិនប្រាកដភត្តាហារក៏មិនប្រាកដ ( ដល់ខ្ញុំ ) សោះ ខ្ញុំព្រះអង្គនោះឯងតែងទៅកាន់ឈាបនដ្ឋានយំរកថា ឪកូនតូចមាសថ្ងៃ អ្នកនៅក្នុងទីណា ឪកូនតូចមាសថ្ងៃ អ្នកនៅក្នុងទីណា ដូច្នេះ ទើបព្រះសមណគោតមត្រាស់ថា ដំណើរនុ៎ះយ៉ាងនេះមែន គហបតី ដំណើរនុ៎ះយ៉ាងនេះមែន គហបតី ពិតមែនគហបតី សេចក្ដីសោកស្ដាយ ខ្សឹកខ្សួលសេចកី្ដទុក្ខ តូចចិត្ត ការចង្អៀតចង្អល់ចិត្ត តែងកើតមកអំពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើត ( កាលបើព្រះមានបុណ្យទ្រង់ត្រាស់យ៉ាងនេះខ្ញុំក៏ទូលតបវិញ ) ថាបពិត្រព្រះអង្គដ៏ចំរើន ធម្មតាហេតុនៃសេចកី្ដស្រឡាញ់នោះឯង នឹងមានដូច្នោះមែន សេចកី្ដសោកស្ដាយ ខ្សឹកខ្សួល សេចកី្ដទុក្ខ តូចចិត្ត ការចង្អៀតចង្អល់ចិត្ត កើតមកអំពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើតមែន បពិត្រព្រះអង្គដ៏ចំរើន តែតាមដែលពិតនោះ សេចកី្ដត្រេកអរនឹងសេចកី្ដរីករាយ តែងកើតមកពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើតដូច្នេះ នែអ្នកដ៏ចំរើនទាំងឡាយ កាលនោះខ្ញុំក៏មិនត្រេកអរ មិនឃាត់ហាមនូវភាសិតរបស់ព្រះសមណគោតមទេ ខ្ញុំក្រោកចាកអាសនៈហើយក៏ចេញមក ។ ពួកអ្នកលេងល្បែងភ្នាល់ឆ្លើយតបថាបពិត្រលោកគហតី ពាក្យដែលលោកមានប្រសាសន៍នុ៎ះ ត្រូវហើយបពិត្រលោកគហបតី ពាក្យដែលលោកមានប្រសាសន៍នុ៎ះ ត្រូវហើយបពិត្រលោកគហបតី សេចកី្ដត្រេកអរនឹងសេចកី្ដរីករាយ តែងកើតមកអំពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើតដោយពិត ។ ទើបគហបតីនោះគិតថា ពាក្យរបស់អាត្មាអញ សមគ្នានឹងពួកអ្នកលេងល្បែងភ្នាល់ ដូច្នេះ រួចក៏ចេញទៅ ។

៣០៨បពិត្រមហារាជ ព្រះអង្គសំគាល់សេចកី្ដនោះថាដូមម្ដេច ដែនកាសីនឹងដែនកោសល តើព្រះអង្គសព្វព្រះរាជឫទ័យឬទេ ។ អើមល្លិកា ដែនកាសីនឹងដែនកោសល ជាទីស្រឡាញ់របស់អញ ម្នាលមល្លិកា ( ព្រោះ ) យើងប្រើប្រាស់ខ្លឹមចន្ទន៍ដែលបានមកអំពីដែនកាសី ទ្រទ្រង់ផ្កាកម្រង គ្រឿងក្រអូបនឹងគ្រឿងលាបផ្សេង ៗ ដោយអានុភាពនៃដែនកាសីនឹងដែនកាសល ។ បពិត្រព្រះមហារាជព្រះអង្គសំគាល់សេចកី្ដនោះថាដូចម្ដេច ( បើ ) ដែនកាសីនឹងដែនកោសលវិបត្តិប្រែប្រួលជាយ៉ាងដទៃទៅ តើសេចកី្ដសោក ខ្សឹកខ្សួល សេចកី្ដទុក្ខ តូចចិត្ត សេចកី្ដចង្អៀតចង្អល់ចិត្តនឹងដល់ព្រះអង្គឬទេ ។ ឪមល្លិកា បើដែនកាសីនឹងដែនកោសល វិបត្តិប្រែប្រួលជាយ៉ាងដទៃទៅសូម្បីជីវិតរបស់អញ ក៏មុខជាវិបត្តិប្រែប្រួលទៅដែរ ចំណង់បើសេចកី្ដសោក ខ្សឹកខ្សួល សេចកី្ដទុក្ខ តូចចិត្ត សេចកី្ដចង្អៀតចង្អល់ចិត្តនឹងខានកើតដល់អញឯណាបាន ។ បពិត្រព្រះមហារាជ ព្រះភគវន្តអរហន្តសម្មាសម្ពុទ្ធអង្គនោះ ទ្រង់ជ្រាប់ច្បាស់ ឃើញច្បាស់ សំដៅយកដំណើរនេះហើយ បានជាសំដែងថា សេចក្ដីសោក ខ្សឹកខ្សួល សេចកី្ដទុក្ខ តូចចិត្ត សេចកី្ដចង្អៀតចង្អល់ចិត្ត តែងកើតឡើងមកអំពីសេចកី្ដស្រឡាញ់ មានសេចកី្ដស្រឡាញ់ជាដែនកើត ។
